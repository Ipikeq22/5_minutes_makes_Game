# 🚀 5分鐘快速做遊戲：打地鼠篇 (5-Min Game Maker: Whack-a-Mole Edition)

這是一個專為國小小朋友設計的 **AI 程式開發教學網站**。透過這個網站，孩子們可以學習如何利用 **GitHub Copilot** 與 **HTML5/CSS3** 技術，在短短 5 分鐘內從零開始打造出一個具備現代感、充滿互動特效的 6x6 打地鼠遊戲。

## ✨ 項目特色 (Features)

- **AI 驅動教學**：利用精確的 AI 提示詞 (Prompt Engineering)，引導 GitHub Copilot 撰寫高品質程式碼。
- **現代感視覺**：採用毛玻璃風格 (Glassmorphism)、全螢幕 6x6 網格與流暢的 CSS 動畫。
- **低門檻開發**：不需要深厚的語法基礎，專注於邏輯思維與素材整合。
- **互動式體驗**：包含角色下載、AI 去背引導、音效整合以及一鍵複製指令功能。

## 🛠️ 技術棧 (Tech Stack)

- **Frontend**: HTML5, CSS3 (Grid & Flexbox), Vanilla JavaScript
- **AI Tools**: GitHub Copilot (Code Generation), Gemini (Image Optimization), Suno (BGM Generation)
- **Deployment**: GitHub Pages

## 📖 如何開始 (How to start)

1. 開啟 [教學網頁](https://ipikeq22.github.io/5-Min-Game-Maker/) (請替換為你的實際網址)。
2. 跟隨網站上的 5 個步驟進行素材下載與 AI 指令複製。
3. 在 VS Code 中貼上代碼，見證奇蹟發生！
