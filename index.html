💻 程式魔術師

現在是最神奇的部分！開啟 VS Code，新建一個資料夾，然後跟著 GitHub Copilot 這樣做：

請幫我撰寫一個「高品質、現代感」的全螢幕 6x6 打地鼠遊戲。 目標是打造一個視覺驚艷、且在任何螢幕下都「不會超出範圍」的網頁應用。 【遊戲流程要求 - 重要！】 - **開始畫面**：遊戲啟動時，必須先顯示一個「開始畫面」遮罩，包含遊戲標題與一個美觀的「開始遊戲」按鈕。 - **點擊開始**：只有在點擊按鈕後，計時器才會開始倒數，且角色才會開始出現。 - **結束畫面**：時間到時，顯示最終分數與「再玩一次」按鈕。 【核心技術要求】 - 使用單一 HTML 檔案。 - **網格尺寸控制**：網格容器寬高設定為 `85vmin`，確保永遠是正方形且契合螢幕。 - **佈局居中**：使用 Flexbox 或 Grid 將容器在畫面上完美居中。 - **禁止捲軸**：CSS 設定 `body { overflow: hidden; }`。 【功能規格】 1. 視覺風格：深色背景搭配毛玻璃 (Glassmorphism) 效果。 2. HUD 介面：左上角顯示分數板，右上角顯示 60 秒倒數。 3. 核心邏輯：隨機閃現角色（正向得 10 分，反派扣 20 分），點擊後需有「浮動分數文字」特效。 4. 支援背景音樂播放與點擊音效。 【變數設定】 正派圖片位置: 反派圖片位置: 背景圖片位置: bgm位置: 點到正派音效位置: 點到反派音效位置: 請包含詳細中文註解。
